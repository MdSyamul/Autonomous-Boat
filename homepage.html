Integrating Machine Learning with IoT for Smart Indoor Farming

Project Background: Indoor farming has been practiced for ages. However, running an indoor farm can be difficult, especially when ensuring each crop has the best possible growing circumstances. Sensing the real-time humidity, temperature, etc. with the help of various sensors and feeding these data to the computer to train the machine (hence, machine learning) can be a big relief for indoor farmers. In this way, a computer can decide when to turn on the pump, switch on the dehumidifier, etc. Hence, this project aims to reduce the efforts of indoor farmers and integrate machine learning with IoT devices to increase the productivity of indoor farming significantly.

Keywords: Automation and Control; Machine Learning; Precision Agriculture.

Project Overview: We will use sensors and IoT devices to gather climate and agricultural data from indoor farming environments. Followed by the application of machine learning algorithms to analyze collected data for predictive modeling and decision-making to automate and optimize various aspects of indoor farming, such as climate control, irrigation, and nutrient delivery.
